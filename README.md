IMDB Movie Analysis

Overview
This project analyzes IMDB movie data using Microsoft Excel to uncover insights on ratings, revenue, and genre trends.

Tools & Techniques
- Microsoft Excel
- PivotTables & Charts
- Data Cleaning & Transformation
- Advanced Functions (INDEX-MATCH, SUMIFS)

Key Insights
- Identified top-performing genres based on ratings and revenue
- Analyzed trends in movie popularity over time
- Compared performance across different categories

Dashboard
Interactive dashboards were created using PivotTables and charts to visualize insights effectively.

Project Files
- `IMDB_Movies.xlsx` → Dataset and analysis

Highlights
- Performed data cleaning and preprocessing
- Built interactive dashboards
- Generated actionable insights for decision-making
